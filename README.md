Транслитератор саха тыла (якутского языка) из кириллицы в алфавит С.А. Новгородова (1917).

🔗 **Репозиторий:** https://github.com/s4m5apu3l/sakha-novgorodov

## Алфавит Новгородова

| Кириллица | Новгородов | | Кириллица | Новгородов |
|-----------|------------|-|-----------|------------|
| А         | a          | | М         | m          |
| Б         | b          | | Н         | n          |
| Г         | g          | | Ҥ         | ŋ          |
| Ҕ         | ʃ          | | Нь        | ɲ          |
| Д         | d          | | О         | ɔ          |
| Дь        | з          | | Ө         | ꭢ          |
| И         | i          | | П         | p          |
| Й         | j          | | Р         | r          |
| К         | k          | | С         | s          |
| Л         | l          | | Һ         | h          |
| Ль        | ʎ          | | Т         | t          |
|           |            | | У         | u          |
|           |            | | Ү         | y          |
|           |            | | Х         | q          |
|           |            | | Ч         | c          |
|           |            | | Ы         | ɯ          |
|           |            | | Э         | e          |

Дифтонги: ꭠ (ыа), ꭡ (иэ), ꭣ (уо), w (үө)

Долгота гласных и согласных обозначается `:` — например аа → a:, мм → m:
